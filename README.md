# salala_Ecommerce_WebApp

* Live Demo : https://sudhisumesh.github.io/javaScript-Ecommerce-website/

This project is an interactive and responsive e-commerce website that allows users to browse and purchase products. The website features efficient add-to-cart functionality using local storage, a fully responsive multi-page layout using Bootstrap 5, and comprehensive form validations to ensure accurate user inputs.
 
* build a responsive interactive UI for a e-commerce mobile selling platform
* use local storage to store cart Items
* when we click add to cart the product will added to cart dinamically also change the quatity and price dinamically
* also we can delete item from the cart
* added validation for forms 
 
 ## Technologies Used
* HTML5
* CSS
* JavaScript
* Bootstrap 5
* jQuery
  
##  Installation

* Clone the repository.
* Open the index.html file in your browser.

## Usage
* Browse Products
To browse products, navigate to the “Products” page. You can filter products by category or search for specific products using the search bar.

* Add Products to Cart
To add products to your cart, click the “Add to Cart” button on the product page. You can view your cart by clicking the cart icon in the top right corner of the page.

* Checkout
To checkout, navigate to the “Cart” page and click the “Checkout” button. You will be prompted to enter your shipping and payment information.
